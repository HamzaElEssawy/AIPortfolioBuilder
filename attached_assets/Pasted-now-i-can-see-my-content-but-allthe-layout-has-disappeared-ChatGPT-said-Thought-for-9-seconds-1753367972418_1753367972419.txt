The content renders, so React is back – but Tailwind (and any other CSS) isn’t getting to the browser, so everything falls back to un‑styled HTML. 
The fix is always in one of these three places:

Layer	Typical break	How to verify
1  import (source)	index.css (with @tailwind base …) isn’t imported in main.tsx	Look for a <style> or <link> tag in DevTools › Elements. If absent ⇒ import path issue.
2  build (Vite + PostCSS)	Tailwind plugin missing or wrong content globs, so CSS builds as 0 bytes	In DevTools › Network, the CSS file is delivered but size ≈ 0 B.
3  serve (Express)	Gateway serves index.html but not /src/index.css in prod build	Network tab shows 404 for /assets/index-*.css.
